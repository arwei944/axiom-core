# LangChain Integration

This guide describes how to connect Axiom cells and signals with LangChain agents and chains.

## Concept Mapping

| Axiom Concept | LangChain Concept |
|---------------|-------------------|
| `Cell` | Agent / Chain |
| `Signal` | Message / Event |
| `Witness` | Audit Log / Trace |
| `Guard` | Safety Check / Validator |

## Setup

Add the Axiom runtime as a tool inside your LangChain agent:

```python
from langchain.tools import BaseTool
import requests

class AxiomCellTool(BaseTool):
    name = "axiom_cell"
    description = "Send a signal to an Axiom cell"

    def _run(self, signal: str) -> str:
        resp = requests.post(
            "http://localhost:8080/cells/my-cell/send",
            json={"signal": signal},
        )
        return resp.text
```

## Example

```python
from langchain.agents import initialize_agent, AgentType
from langchain.llms import OpenAI

tools = [AxiomCellTool()]
agent = initialize_agent(tools, OpenAI(), agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION)

result = agent.run("Send a greeting signal to the hello cell")
```

## Observability

Use `axm trace view <trace_id>` to inspect the causal chain generated by LangChain interactions.
